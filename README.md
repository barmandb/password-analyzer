# Password Security Analyzer

A Flask-based password security tool that checks password strength, detects weak patterns, estimates crack time, verifies breach exposure, and generates strong passwords.

## Features
- Password strength meter
- Weak pattern detection
- Estimated crack time
- Breach check using k-anonymity API
- Secure password generator
- Modern UI dashboard

## Tech Stack
- Python
- Flask
- requests
- HTML
- CSS
- JavaScript

## Usage

```bash
pip install -r requirements.txt
python app.py
